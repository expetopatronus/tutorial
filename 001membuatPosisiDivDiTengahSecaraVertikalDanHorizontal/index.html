Membuat Posisi Div di Tengah Secara Vertikal dan Horizontal

Gimana sih caranya agar posisi suatu element selalu supaya berada di tengah secara vertikal maupun horizontal?

Me: Hmm, good question.

Setau Me ini ada dua teknik. Yaitu teknik display: flex dan position: absolute. Mari kita coba satu-satu.

A. display: flex

Pertama display: flex adalah teknik yang paling sering digunakan oleh pengembang website yang skillnya sudah tinggi. Jika ingin tahu tentang display: flex kalian bisa mengunjungi w3school.com.

1. Buat file html
Isi kan file html yang telah kalian buat dengan kode berikut: 

<!DOCTYPE html>
<html>
<head>
	<title>Tutorial</title>
	<link rel="stylesheet" type="text/css" href="style.css">
	<meta charset="utf-8">
	<meta name="viewport" content="width=device-width,initial-scale=1">
</head>
<body>
	<div class="kotak">
		<div class="lingkaran"></div>
	</div>
</body>
</html>

2. Buat file style.css di folder yang sama
Kita tandai dulu mana yang class kotak dan mana yang kelas lingkaran dengan cara memberi warna dan ukuran. Berikut untuk memberikan style pada class kotak: 

.kotak {
	height: 400px;
	width: 400px;
	background: yellow;
}

berikut hasilnya:

Sekarang buat style untuk yang lingkaran: 

.kotak {
	height: 400px;
	width: 400px;
	background: yellow;
}

.lingkaran {
	height: 100px;
	width: 100px;
	background: blue;
	border-radius: 50%;
}

berkut hasilnya:

3. Ditengah secara horizontal
Untuk menengahkan elemen secara horizontal kita harus menambah display: flex di class yang luar, yakni class kotak: 

.kotak {
	height: 400px;
	width: 400px;
	background: yellow;
	display: flex;
}

Sekarang kita posisikan ke tengah secara horizontal, dengan menambahkan justify-content: center; di class kotak: 

.kotak {
	height: 400px;
	width: 400px;
	background: yellow;
	display: flex;
	justify-content: center;
}

Jika kalian cek maka posisi lingkaran sudah berada di tengah secara horizontal.

4. Secara vertical
Sekarang kita buat agar berada di tengah secara vertical, dengan cara di class kotak tambahkan align-items: center; 

.kotak {
	height: 400px;
	width: 400px;
	background: yellow;
	display: flex;
	justify-content: center;
	align-items: center;
}
