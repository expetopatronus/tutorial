Sekarang mari kita bahas cara membuat posisi div di tengah secara vertikal dan horizontal menggunakan teknik position: absolute;. Apa sih kegunaan dari position: absolute;? Yaitu untuk meletakkan posisi secara bertumpuk. Untuk lebih lengkapnya kalian bisa cek di w3school.com. Untuk tutorial ini kita akan fokus ke topik.




B. position: absolute


1. Buat file index.html dan style.css dalam direktori yang sama.

index.html :

<!DOCTYPE html>
<html>
<head>
	<title>Tutorial</title>
	<link rel="stylesheet" type="text/css" href="style.css">
	<meta charset="utf-8">
	<meta name="viewport" content="width=device-width,initial-scale=1">
</head>
<body>
	<div class="kotak">
		<div class="lingkaran"></div>
	</div>
</body>
</html>

style.css: 

.kotak {
	height: 400px;
	width: 400px;
	background: yellow;
}

.lingkaran {
	height: 100px;
	width: 100px;
	background: blue;
	border-radius: 50%;
}

Mirip dengan cara pertama bukan? Next..



2. Tambahkan properti position: absolute di class kotak dan class lingkaran

Maka kode style.css jadi seperti berikut: 

.kotak {
	height: 400px;
	width: 400px;
	background: yellow;
	position: absolute; /* <- tambahkan*/
}

.lingkaran {
	height: 100px;
	width: 100px;
	background: blue;
	border-radius: 50%;
	position: absolute; /* <- tambahkan*/
}

Tunggu dulu. Memang belum ada perubahan, hehe



3. Tambahkan jarak atas dan kiri class lingkaran
Kita akan menggunakan satuan persen untuk kasus ini.

Berikut kode di class lingkaran: 

.lingkaran {
	height: 100px;
	width: 100px;
	background: blue;
	border-radius: 50%;
	position: absolute;
	top: 50%; /* <- tambahkan*/
	left: 50%; /* <- tambahkan*/
}

Kok belum ke tengah bang? Masih ada satu langkah lagi. Ini yang terakhir, semoga saja, hehe



4. Kita geser -50% dari ukuran object

Gunakan transform: translate(-50%, -50%); maka elemen akan geser ke kiri dengan jarak setengah dari lebar elemen, dan geser ke atas sebesar setengah dari tinggi elemen.

Berikut kodenya: 

.lingkaran {
	height: 100px;
	width: 100px;
	background: blue;
	border-radius: 50%;
	position: absolute;
	top: 50%; 
	left: 50%;
	transform: translate(-50%, -50%); /* <- tambahkan*/
}

Bisa dilihat kan bedanya. Jika kita menggunakan teknik display: flex; maka elemen tetap sejajar. Sedangkan jika menggunakan teknik position: absolute; elemen akan bertumpuk.
